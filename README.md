# Helmet-Detection-for-Bike-Riders-using-YOLO5
A real-time *computer vision project* that detects whether a *bike rider is wearing a helmet or not* using the *YOLOv5 deep learning model*.  
This project aims to enhance *road safety* by providing an automated solution for traffic monitoring systems.

## Project Overview
Two-wheeler riders not wearing helmets are at high risk of head injuries during accidents.  
Manual monitoring by traffic police is difficult and time-consuming.  
This project automates the process using *YOLO (You Only Look Once)* ‚Äî a state-of-the-art, real-time object detection algorithm.
